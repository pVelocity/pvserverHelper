### ``setupLogger(jsapi, [enableConsoleInfo])``
Setups a logger for logging messages.
- `jsapi` `<Object>`
- `enableConsoleInfo` `<Boolean>` : Optional, default is ``false``

```js
pvh.setupLogger(jsapi);
```
